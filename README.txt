1. Abra 2 terminais e vá para a pasta onde encontra o código do programa;

2. Com um dos terminais, compile o código digitando o comando "make";

3. Inicie o servidor em um dos terminais através do comando "./server {1} {2}", sendo o parâmetro 1 o tipo de conexão ("v4" para IPV4 e "v6" para IPV6), e o parâmetro 2 o número da porta;

4. Inicie o cliente no outro terminal através do comando "./client {1} {2} start", sendo o parâmetro 1 o IP ("127.0.0.1" para IPV4 e "::1" para IPV6), e o parâmetro 2 o número da porta;

5. Digite os comandos no terminal do cliente para jogar;

6. Para encerrar o programa em ambos os terminais, basta enviar o comando "quit" do cliente para o servidor;
